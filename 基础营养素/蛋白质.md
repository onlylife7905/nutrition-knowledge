# 蛋白质

## 基本概念
蛋白质是由氨基酸组成的大分子有机物，是生命的物质基础。

## 生理功能
- 构成和修复身体组织
- 合成酶、激素、抗体等活性物质
- 供能（1克蛋白质 = 4千卡）
- 维持渗透压和酸碱平衡

## 每日推荐摄入量
| 人群 | 推荐量 |
|------|--------|
| 成年男性 | 65g/天 |
| 成年女性 | 55g/天 |
| 孕妇（中期） | +15g/天 |
| 老年人 | 1.0-1.2g/kg体重 |

## 优质蛋白质来源
- **动物性**：鸡蛋、牛奶、鱼、禽肉、瘦肉
- **植物性**：大豆及豆制品（含全部必需氨基酸）

## 必需氨基酸（9种）
亮氨酸、异亮氨酸、缬氨酸、赖氨酸、蛋氨酸、苯丙氨酸、色氨酸、苏氨酸、组氨酸

## 注意事项
- 蛋白质摄入过多会加重肾脏负担
- 建议动植物蛋白合理搭配
- 蛋白质互补原则：多种食物混合食用可提高利用率
