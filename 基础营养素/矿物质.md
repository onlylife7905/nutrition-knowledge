# 矿物质

## 概述
矿物质（无机盐）是人体必需的无机营养素，约占体重的4%。

## 主要矿物质
| 矿物质 | 功能 | 推荐摄入量 | 食物来源 |
|--------|------|------------|----------|
| 钙 | 骨骼牙齿、凝血、肌肉收缩 | 800-1000mg/天 | 奶制品、豆腐、绿叶菜 |
| 铁 | 血红蛋白合成、氧气运输 | 男12mg/女20mg | 红肉、动物肝脏、黑木耳 |
| 锌 | 免疫、生长发育、味觉 | 男12.5mg/女7.5mg | 牡蛎、肉类、坚果 |
| 碘 | 甲状腺激素合成 | 120μg/天 | 海产品、碘盐 |
| 钾 | 血压调节、心脏功能 | 2000mg/天 | 香蕉、土豆、豆类 |
| 钠 | 体液平衡、神经传导 | <2000mg/天 | 食盐（需限制） |
| 镁 | 能量代谢、骨骼健康 | 330mg/天 | 坚果、绿叶蔬菜、全谷物 |

## 重要提示
- **缺铁性贫血**：中国最常见的营养缺乏病，女性及儿童高发
- **骨质疏松**：与钙和维生素D摄入不足有关
- **高血压**：与高钠低钾饮食密切相关，建议限盐
- 矿物质间存在相互作用（如钙与铁吸收竞争），需合理搭配
