# 糖尿病饮食管理

## 基本原则
- 控制总热量，维持理想体重
- 稳定血糖，减少波动
- 均衡营养，预防并发症

## 饮食要点

### 碳水化合物
- 占总热量的 45-60%
- 优先选择低GI（升糖指数）食物
- 避免精制糖、含糖饮料

### 低GI食物推荐
| 食物 | GI值 |
|------|------|
| 燕麦 | 55 |
| 糙米 | 56 |
| 全麦面包 | 50 |
| 绿叶蔬菜 | <15 |
| 苹果 | 36 |

### 蛋白质
- 占总热量的 15-20%
- 肾功能正常者：0.8-1.0g/kg体重
- 优选鱼类、禽肉、豆类

### 脂肪
- 占总热量的 25-35%
- 减少饱和脂肪，增加不饱和脂肪
- 避免反式脂肪酸

## 进餐建议
- 定时定量，少量多餐
- 先吃蔬菜，再吃蛋白质，最后吃主食
- 细嚼慢咽，延缓血糖上升
- 运动后及时补充适量碳水

## 禁忌食物
- 白糖、蜂蜜、糖果
- 含糖饮料、果汁
- 精白米面（过量）
- 油炸食品

## 监测指标
- 空腹血糖：4.4-7.0 mmol/L
- 餐后2小时血糖：< 10.0 mmol/L
- 糖化血红蛋白（HbA1c）：< 7%
